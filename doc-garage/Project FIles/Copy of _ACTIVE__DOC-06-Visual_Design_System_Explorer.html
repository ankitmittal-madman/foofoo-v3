Gesture rules
Not Today ↑
Long-press + drag up 80px. Skips this dish today. Exponential decay — dish returns to normal scoring after ~7 days.
Never ↓
Long-press + drag down 80px. Permanently removes dish from all future suggestions. Dish goes to Never List in Profile.
Motion spec
Long-press trigger: 400ms
Card lift: scale(1.02) over 150ms
Zone slide: translateY + opacity over 150ms
Never zone color: #E53935
Not Today color: #F59E0B
Haptic on trigger: medium impact

Onboarding principles
✅ One question per screen — never group multiple questions
✅ Visual chips — not text dropdowns or radio buttons
✅ Skip always visible — top-right corner, small, not prominent
✅ CTA activates on selection — grayed out until something is chosen
✅ Progress bar — thin, minimal, shows position without overwhelming
❌ Never a back button that loses data
❌ Never a required field asterisk
