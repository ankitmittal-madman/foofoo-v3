## 1. Purpose

FooFoo has, since `AGR-005`/`AGR-006`, had a correct instrument for ratifying an individual Founder-level decision (an "AGR" — Architecture Governance Record) but no single place that lists every decision awaiting, or having received, that ratification. Decisions have instead accumulated inside whichever document happened to surface them: a work package's own reconciliation notes (`DCR-8D-01`), a draft specification's self-declared open item (`AD-01`), a research batch's gap tracker (`B4-GAP-001`), an audit's own findings table (`FD-01`–`FD-13`). Each of those is a legitimate primary source — none of them is discoverable from the others.

This Book exists to solve exactly that discoverability problem, and nothing else. It does not re-argue any decision, does not re-audit the repository, and does not contain any evidence that isn't already written down somewhere else, cited here.

## 2. How to use this document

- **If you are about to implement something and suspect it depends on an unratified assumption:** check §5 (Pending) and §6 (Deferred) before writing code.
- **If you are the Founder reviewing what needs a signature:** §9 is the complete sign-off checklist.
- **If you just ratified something:** add it to §4, assign it the next `AGR-NNN` number, remove it from §5/§6, and add one line to §10.
- **Never copy a decision's full reasoning into this Book.** Cite the primary source. If a decision has no primary source document yet (this happens — see FD-01/FD-06/FD-07 below), say so explicitly rather than inventing one.

## 3. Decision Lifecycle

```
Surfaced (found during any WP/audit/batch, recorded at its point of origin)
   │
   ▼
Pending  ──────────────► Deferred (explicit, dated, reasoned deferral — not silence)
   │                          │
   ▼                          ▼
Approved (assigned an AGR-NNN, Founder-signed)      remains Deferred until revisited
   │
   ▼
Superseded (only when a later ratified decision explicitly replaces it — never silently)
```

A decision only leaves "Pending" by moving to "Approved" or "Deferred" — never by being forgotten. If this Book is ever reopened and a Pending item from months ago is still Pending, that is a signal to escalate it, not evidence it stopped mattering.
